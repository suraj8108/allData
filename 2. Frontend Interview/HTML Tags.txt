HTML Semantic
	-> In HTML, semantics refers to the use of HTML tags that convey the meaning and structure of the content they enclose. 
	-> Semantic HTML elements clearly describe their purpose both to the browser and the developer, making the code more readable and accessible.
	
	-> Key Points of HTML Semantic
		1. Improved Accessibility: Help screen readers and other assistive technologies understant the structure and meaning of web content. Helps disable people.
		2. Better SEO
		3. Enhanced Readability
		4. Standardization
